# Convertisseur de Température en Temps Réel


## Fonctionnalités

- **Mise à jour en temps réel** : Les résultats changent instantanément quand vous tapez
- **3 échelles de température** : Celsius (°C), Fahrenheit (°F), Kelvin (K)
- **Design moderne** : Effet de verre (glassmorphism), animations fluides
- **Responsive** : S'adapte parfaitement aux mobiles, tablettes et ordinateurs
- **Performant** : JavaScript pur, pas de bibliothèques externes
- **Précis** : Arrondi à 4 décimales pour des résultats précis

## Comment ça fonctionne ?

### La logique de conversion (explication simple)

Le convertisseur utilise **le Celsius comme unité centrale** pour simplifier les calculs.
